## 🧠 What is Spotifyt?

Spotifyt is a desktop music player that brings a clean, Spotify-style interface to YouTube Music. Log in with your Google account and stream your favorite music — no compromises, no switching tabs, no browser clutter.

If you have YouTube Premium? No ads. Just music.

> ⚠️ **Early Alpha** — This is v1.0.0-beta. It works, but barely. You've been warned (lovingly).

## ✨ Features

- 🎵 **Stream from YouTube Music** — all the music, familiar UI
- 📁 **Playlist & Album support** — browse and play your collections
- 🔐 **YouTube account login** — your library, your preferences
- 🚫 **Ad-free with YouTube Premium** — if you have it, Spotifyt respects it
- 📋 **Queue viewer** — see what's coming up next

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed
- A YouTube / Google account (for login)
- YouTube Premium (optional, for ad-free experience)

### Installation

```bash
# Clone the repository
git clone https://github.com/iisuperboyii/spotify-yt-personal-app

# Navigate into the project
cd spotify-yt-personal-app

# Install dependencies
npm install

# Run the app
npm start
```

Then log in with your YouTube account inside the app and you're good to go. 🎧
